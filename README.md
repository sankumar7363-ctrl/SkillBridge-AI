# 🚀 SkillBridge AI

An AI-powered career guidance platform that helps students and job seekers analyze their resumes, identify skill gaps, evaluate ATS compatibility, and receive personalized career roadmaps.

## ✨ Features

- 📄 Resume Upload & Parsing
- 🤖 AI-Powered Career Guidance
- 📊 ATS Compatibility Score
- 🎯 Skill Gap Analysis
- 🛣️ Personalized Learning Roadmap
- ⚡ n8n Workflow Automation
- 💻 Responsive User Interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sankumar7363-ctrl/SkillBridge-AI.git
```

Go to the project folder

```bash
cd SkillBridge-AI
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

### Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
